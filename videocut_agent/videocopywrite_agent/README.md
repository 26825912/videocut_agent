# VideoCopywrite Agent

爆款视频仿写智能体 - 学习成功案例快速创作新视频

## 📝 功能概述

VideoCopywrite Agent 是一个专业的爆款视频仿写智能体，可以分析成功视频的创作模式，并基于该模式创作新主题的短视频。通过学习爆款视频的叙事结构、节奏和风格，帮助创作者快速生产高质量内容。

## 🎯 核心能力

### 1. 爆款视频分析
- **内容分析**: 深度分析爆款视频的文案结构
- **风格提取**: 识别视频的叙事风格和节奏
- **成功要素**: 总结视频受欢迎的关键因素

### 2. 脚本仿写
- **模式迁移**: 将成功模式应用到新主题
- **风格保持**: 保留原视频的创作风格
- **内容创新**: 在保持风格的同时创作新内容

### 3. 全流程生成
- **自动化处理**: 从脚本到视频的完整流程
- **质量保证**: 确保仿写视频保持高质量
- **快速产出**: 比从零开始更快速高效

### 4. 多语言支持
- **默认英文**: 未指定语言时使用英文
- **中英切换**: 支持中文和英文内容创作
- **本地化**: 适应不同语言的表达习惯

## 🔧 技术架构

### LangGraph 仿写流程

```
START → 视频分析 → 脚本仿写 → 语音生成 → 字幕生成 → 素材匹配 → 视频合成 → END
           ↓           ↓          ↓          ↓          ↓          ↓
       分析工具     仿写工具    TTS工具   字幕工具   搜索工具   合成工具
```

### 完整工作流程

1. **阶段1: 视频分析**
   - 输入: 爆款视频链接
   - 处理: 提取视频文案和结构
   - 输出: 视频分析报告

2. **阶段2: 脚本仿写**
   - 输入: 分析结果、新主题、时长
   - 处理: 基于原视频风格创作新脚本
   - 输出: 仿写后的视频脚本
   - **默认使用英文**（除非明确指定中文）

3. **阶段3: 语音生成**
   - 输入: 仿写脚本
   - 工具: TTS
   - 输出: 音频文件

4. **阶段4: 字幕生成**
   - 输入: 音频文件
   - 工具: ASR + 字幕生成
   - 输出: ASS字幕文件

5. **阶段5: 素材检索**
   - 输入: 脚本内容和时间轴
   - 处理: 提取关键词并搜索素材
   - 输出: 匹配的视频/图片素材

6. **阶段6: 视频合成**
   - 输入: 素材、音频、字幕
   - 处理: 合成最终视频
   - 输出: 完整的仿写视频

### 系统提示词

```
你是一个爆款视频仿写助手，可以根据用户输入的视频链接、仿写主题和时长范围仿写一个新的视频

用户没有指定的话默认使用英文

1. 接到指令之后，确定是否有视频链接、仿写主题、时长范围信息
2. 先仿写爆款视频文案，生成文案文件（没有指定语言的情况下默认使用英文）
3. 根据爆款文案，生成语音文件
4. 根据语音文件，生成字幕文件
5. 把爆款文案文件和语音文件，生成对应的素材检索关键词和时间轴
6. 根据关键词和时间轴，搜索并剪辑素材
7. 合成最终视频
```

## 📝 注意事项

1. **视频链接**:
   - 需要提供可访问的视频链接
   - 支持 TikTok、YouTube、抖音等平台
   - 确保链接有效且视频可访问

2. **语言默认**:
   - ⚠️ **重要**: 默认使用英文
   - 必须明确说明"使用中文"才会生成中文视频
   - 仅支持中文和英文

3. **主题适配**:
   - 新主题应与原视频风格匹配
   - 过于不同的主题可能效果不佳
   - 建议选择同类型的主题进行仿写

4. **版权问题**:
   - 仿写是学习风格，不是抄袭内容
   - 确保创作的是原创内容
   - 避免直接复制原视频的独特元素

5. **质量控制**:
   - 仿写结果可能需要人工调整
   - 建议对比原视频检查风格是否一致
   - 多次尝试选择最佳结果

6. **时长匹配**:
   - 建议仿写视频时长与原视频接近
   - 时长差异过大可能影响风格还原
   - 可以适当调整但不宜偏离太多

## 💡 使用技巧

### 选择合适的参考视频

**好的参考视频**:
- 播放量高、互动率高
- 内容结构清晰
- 叙事节奏明确
- 风格独特但可迁移

**避免的参考视频**:
- 依赖特定场景或道具
- 个人化色彩过重
- 风格过于独特难以复制

### 优化仿写效果

1. **详细描述新主题**: 提供足够的主题细节
2. **保持时长接近**: 与原视频时长差异不要太大
3. **明确目标受众**: 说明面向什么群体
4. **指定语言**: 明确使用中文还是英文
5. **多次尝试**: 生成多个版本选择最佳

## ⚠️ 伦理和法律考虑

1. **内容原创性**:
   - 仿写是学习风格，不是复制内容
   - 确保生成的内容是原创的
   - 不要直接抄袭原视频的文案

2. **版权尊重**:
   - 不要仿写有明显个人风格的创作者
   - 避免使用原视频的独特元素
   - 尊重原创者的知识产权

3. **平台规则**:
   - 遵守各平台的内容政策
   - 避免批量发布相似内容
   - 保持内容多样性
